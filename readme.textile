h2. CakePHP Cycle Css

Cylce Css is a helper that will output alternating text, its very usefull for doing alternating row colours in a table.

h3. So how do I use it?

<pre>
<table>
 <? foreach(array(1,2,3) as $item) : ?>
    <tr class="<?= $cycle->css() ?>">
      <td><?= $item ?></td>
    </tr>
 <? endforeach ?>
</table>
</pre>

  Output is as follow

<pre>
<table>
  <tr class="odd">
   <td>1</td>
  </tr>
  <tr class="even">
    <td>2</td>
  </tr>
  <tr class="odd">
    <td>3</td>
  </tr>
 </table>
</pre>

If you wanted to show the class tags also

<pre>
<table>
 <? foreach(array(1,2,3) as $item) : ?>
    <tr <?= $cycle->css(array('odd','even'), true) ?>>
      <td><?= $item ?></td>
    </tr>
 <? endforeach ?>
</table>
</pre>

Output is as follow

<pre>
<table>
  <tr class="odd">
   <td>1</td>
  </tr>
  <tr class="even">
    <td>2</td>
  </tr>
  <tr class="odd">
    <td>3</td>
  </tr>
</table>
</pre>

Make sense?
